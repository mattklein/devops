Use cases

AMI management

  1. Create an AMI from a definition
    • Create an instance from the definition
    • Snap an AMI from the instance
    • Terminate the instance

  2. Create an instance from an AMI
    • Obtain the list of AMIs from the ops registry
    • Invoke a method on the AMI: create an instance

  3. Activate an instance

Ops registry

  1. Obtain the AMIs
    • Query for the AMIs; optional filter criteria:
      • AMI name
      • AMI ID
    • Operate on an AMI
      • Create a new instance

        • (Also have a way to do this without first fetching the AMI)
